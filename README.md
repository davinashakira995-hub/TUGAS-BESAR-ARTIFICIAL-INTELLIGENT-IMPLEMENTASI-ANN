# TUGAS-BESAR-ARTIFICIAL-INTELLIGENT-IMPLEMENTASI-ANN
Repository ini dibuat untuk memenuhi persyaratan pengumpulan UAS Tugas Besar Implementasi ANN from Sctratch

Folder source Code, ada 2 file yaitu scratch.ipynb yang berisi implementasi ANN from scratch dan keras.ipynb yang berisi implementasi ANN dengan tensorflow keras, bisa di running menggunakan kernel venv(Python3.11)

folder doc berisi laporan mengenai tugas yang membahas mengenai pembahasan masalah, implementasi, hasil pengujian, dll.
